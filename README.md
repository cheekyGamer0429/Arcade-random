# Arcade-random

An arcade based project using keys from a random password generator. It focuses on playing games to earn points to be later used in purchasing prizes. This project is for entertainment purposes.

## Technologies Used

- Python
  - Python is the main coding language used.
    
- secrets library
  - Used to make random passwords and it to be more secure.
    
- string library
  - Used to get all available characters.
